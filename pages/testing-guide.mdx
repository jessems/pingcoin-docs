# Testing Guide

Master the art of coin testing with these advanced techniques and best practices.

## The Perfect Setup

### Environment
| Factor | Ideal | Acceptable | Avoid |
|--------|-------|------------|-------|
| **Noise level** | Silent room | Quiet background | Music, TV, appliances |
| **Surface** | Felt pad or fingertip | Soft cloth | Hard surfaces (wood, glass) |
| **Distance** | Phone 6-12 inches away | Up to 18 inches | Too close (distortion) or far (weak signal) |

### The Ideal Testing Station

Create a dedicated testing spot:
1. **Felt pad** — A 6"x6" jewelry pad works perfectly
2. **Metal striker** — Another coin or small metal rod
3. **Good lighting** — To inspect the coin visually too
4. **Quiet space** — Away from HVAC, refrigerators, traffic noise

## Tapping Technique

The tap is everything. A poor tap produces unreliable results.

### The Correct Tap

```
        ↓ Strike here
        ●
    ┌───────────┐
    │           │    ← Edge strike produces 
    │   COIN    │       clearest ring
    │           │
    └───────────┘
```

1. **Strike the edge** — About 1/3 from the rim
2. **Single clean contact** — No bouncing or double-taps
3. **Light to medium force** — Enough to ring, not so hard it rattles
4. **Let go immediately** — Don't touch the coin after striking

### Common Mistakes

| Mistake | Problem | Fix |
|---------|---------|-----|
| Center strike | Muted, unclear frequencies | Strike near edge |
| Too hard | Clipping, distortion | Lighter touch |
| Multiple taps | Interfering frequencies | One clean tap |
| Holding the coin | Dampens vibration | Balance on fingertip or felt |
| Plastic striker | Wrong harmonics | Use metal (coin, rod) |

## Holding Methods

How you hold the coin affects its vibration.

### Method 1: Fingertip Balance (Recommended)
- Balance coin flat on your fingertip
- Minimal contact = maximum ring
- Best for smaller coins

### Method 2: Felt Pad
- Lay coin flat on soft felt
- Slightly dampens vibration but consistent
- Best for larger, heavier coins

### Method 3: Edge Balance
- Stand coin on edge against a soft support
- Allows both faces to vibrate freely
- Tricky to set up but produces clean results

### Method 4: Pinch Hold (Not Recommended)
- Holding coin between fingers
- Significantly dampens vibration
- Avoid unless no other option

## Interpreting Results

### Strong Pass ✅
- Multiple frequency peaks at expected positions
- Clean, sustained ring detected
- High confidence match

**Action:** Coin is very likely authentic

### Weak Pass ✅
- Frequencies match but with lower confidence
- May indicate suboptimal testing conditions
- Or a legitimate variant with slightly different specs

**Action:** Retest with better technique. If consistent weak pass, likely authentic but verify with other methods for high-value coins.

### Fail ❌
- Significant frequency mismatch
- Unexpected peaks or missing peaks
- Decay pattern inconsistent with material

**Action:** Investigate. Check if correct coin type was selected. Retest. If consistent fail, coin may be counterfeit.

### Inconclusive ⚠️
- Unable to extract clear frequency data
- Usually indicates testing environment or technique issues

**Action:** Fix your setup and try again. See troubleshooting below.

## Troubleshooting Results

### "My authentic coin keeps failing"

1. **Verify coin selection** — Is the exact coin/weight selected?
2. **Check for damage** — Bent, cleaned, or repaired coins may test differently
3. **Environmental factors** — Test in quieter location
4. **Microphone issues** — Try recording with a different app to verify mic works
5. **Coin variant** — Some years/mints have slight spec differences

### "Results are inconsistent"

1. **Standardize your technique** — Same tap location, same force
2. **Same environment** — Acoustics matter more than you think
3. **Clean the coin** — Dirt and oils dampen vibration
4. **Check phone position** — Keep consistent distance

### "Always getting inconclusive"

1. **Background noise** — Find a quieter spot
2. **Microphone permission** — Is the app allowed to record?
3. **Microphone blocked** — Check your phone case isn't covering it
4. **Try test recordings** — Record yourself talking to verify mic

## Advanced Techniques

### Multiple Test Protocol

For high-value purchases, use this protocol:

1. **Test 3 times** with standard technique
2. **Test once** with coin on fingertip
3. **Test once** with coin on felt
4. **Record results** — Look for consistency

If 4-5 tests pass → High confidence authentic  
If mixed results → Investigate further  
If majority fail → Concern warranted

### Comparative Testing

If you have an authenticated coin of the same type:

1. Test the known-authentic coin first
2. Note the frequency spectrum
3. Test the unknown coin
4. Compare spectra visually

Similar patterns = good sign  
Notably different = concern

### Temperature Considerations

Metal properties change slightly with temperature:
- Test coins at room temperature (65-75°F / 18-24°C)
- Very cold or hot coins may produce slightly shifted frequencies
- Let coins acclimate for 15+ minutes if stored in extreme conditions

## Documentation

### Keep Records

For valuable collections:
- Screenshot test results
- Note date, conditions, result
- Store with purchase receipts
- Retest periodically (especially after purchases at new venues)

### Building a Reference Set

If you buy frequently:
- Keep one verified-authentic coin of each type
- Use it as your personal reference standard
- Compare new purchases against your reference

## Combining with Other Tests

Pingcoin is most powerful when combined with:

| Test | Catches | Miss |
|------|---------|------|
| **Visual inspection** | Wrong design, poor quality fakes | High-quality visual counterfeits |
| **Weight test** | Wrong metal (lead, copper) | Tungsten (same density as gold) |
| **Dimension test** | Oversized/undersized fakes | Correct-size fakes |
| **Magnet test** | Ferrous metals | Non-magnetic fakes |
| **Ping test (Pingcoin)** | Wrong material, tungsten cores | Unofficial mint with correct metal |

**Pro tip:** A coin that passes ALL these tests has extremely high probability of authenticity.

## Testing Etiquette

### At Coin Shops
- Ask permission before testing in-store
- Explain what you're doing if asked
- Don't waste dealer time testing obvious bullion
- Be discreet — constant pinging annoys everyone

### At Coin Shows
- Busy, noisy environments = suboptimal testing
- Step away from the booth to a quiet area
- Quick pre-test, detailed test at home
- Most sellers won't mind a quick check

### With Private Sellers
- Testing protects both parties
- Offers transparency
- Serious sellers appreciate careful buyers
